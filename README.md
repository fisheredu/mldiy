# mldiy

This repository contains personal notes and experiments about machine learning and related mathematics. It is intended for personal and educational use only.

## Purpose
- Personal learning and reference: notes, worked examples, and short tutorials.
- Educational demonstrations: small snippets to illustrate concepts and derivations.

## Contents
- `docs/` — Markdown source for the site and notes.
- `mkdocs.yml` — MkDocs configuration (local site preview).
- `docs/js/` — JavaScript helpers (MathJax config and helpers for rendering math).

## Local preview
Install `mkdocs` and serve locally:

```bash
pip install mkdocs mkdocs-material
mkdocs serve
# open http://127.0.0.1:8000
```

Alternatively, open individual Markdown files in VS Code and use a Markdown preview extension (e.g., "Markdown Preview Enhanced") for MathJax rendering.
